<p class="px-6">O terminal pode ser um ambiente desconfortável e improdutivo caso você não crie familiaridade com as ferramentas que ela possui. É interessante não apenas conhecer as ferramentas nativas de um sistema linux, como os do projeto GNU, como também as ferramentas adicionais que você pode instalar para facilitar seu dia a dia no terminal. A lista desse post apresenta ferramentas que tanto agregam em agilidade no terminal, como oferecem mais riqueza visual pra tela de texto. Elas servem tanto para quem está comecando, como para aqueles que já tem autonomia, mas podem não conhecer alguns dos itens abaixo.</p>

# 1. tmux

<Slider>
  ![image](/assets/images/posts/tmux.png)
  <VideoFrame id="Yl7NFenTgIo" />
</Slider>

**Bom para:** Gerenciar múltiplas seções de terminal, gestão visual por janelas.

tmux é um multiplexador de terminais. Em outras palavras, ele permite gerenciar múltiplas sessões de terminal por linha de comando e atalhos no teclado. Ele é útil para lidar com mais de uma sessão no mesmo terminal, gerenciar sessões por meio de janelas, mandar sessões abertas para segundo plano, e no geral, ter uma interface amigável para navegar entre processos abertos no terminal. Ele faz ainda mais sentido quando você está limitado à acessar uma máquina remota apenas por SSH. Tmux vai te dar muito mais liberdade.

O tmux tem suporte à plugins, que permitem extender o programa com funções **como pré definir layouts de janelas ([tmuxinator](https://github.com/tmuxinator/tmuxinator))**, para ambientes diferentes entre projetos, e **restaurar o layout anterior mesmo após reiniciar a máquina ([tmux-continuum](https://github.com/tmux-plugins/tmux-continuum))**.

# 2. Nerd fonts

![image](/assets/images/posts/nerd-fonts-list.svg)

**Bom para:** Renderizar icones úteis pela linha de comando.

Nerd fonts não é um programa, mas um pacote de ícones disponibilizado como fontes de texto. São mais de 9 mil ícones inclusos que, ao utilizar como uma de suas fontes para o terminal, é possível visualizar ícones na linha de comando, permitindo programas se tornarem mais ricos visualmente.

Para facilitar o seu uso, a forma que esses ícones são disponibilizados é unindo-os com alguma outra fonte de texto que vocẽ use, como `UbuntuMono`, `Fira Code`, `Hack`. Dessa forma, você só precisa setar a fonte de texto que escolheu, que os ícones já estarão inclusos no terminal.
